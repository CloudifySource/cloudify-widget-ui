widget_recipe_properties

This message allows you to send properties to the recipe.
The widget passes them through to the recipe unaware of their content.
The event looks like this:
{ 'name' : 'widget_recipe_properties', 'data': [ { 'key' : 'someProperty', 'value' : 'someValue' }, { 'key' : 'anotherProperty', 'value' : 'anotherProperty' } ] }
