widget_play

This message allows you to play the widget
{ 'name': 'widget_play' }
In the case the widget is configured as 'Solo Mode', the message needs to also include the ApiKay and SecretKey.
Each cloud type will have a different structure.
{ 'name': 'widget_play', 'widget': { 'executionDetails': { 'EC2': { 'params': { 'apiKey': 'the api key', 'secretKey': 'the secret key' } } } } }
